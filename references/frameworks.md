# Writing Frameworks

Five reusable frameworks. Pick one, fill in the blanks.

---

## 1. Narrative (故事→洞察→行动)

Best for: personal stories, lessons learned, growth arcs.

```
[Hook]    "X年前，我[做了某事/经历了某事]..."
[Context] 当时的情况、状态、面临的挑战
[Conflict] 出了什么问题？困惑/挣扎
[Turn]    关键 moment — 顿悟或改变
[Insight] 我学到了什么？
[Action]  这个教训如何改变了我的行为
[CTA]     你可以如何应用这个洞察
```

Example hook: "我的奶奶经历过大萧条，这是她告诉我的金钱秘密"

---

## 2. Contrast (旧认知 vs 新认知)

Best for: challenging consensus, redefining concepts, cognitive upgrades.

```
[Hook]    "大多数人认为的A其实是错的"
[Old]     描述普遍观点 + 为什么它看起来合理
[Problem] 这个认知带来的问题/局限
[New]     提出新视角/框架
[Proof]   案例、数据、逻辑论证
[Apply]   如何在实际中运用新认知
[CTA]     邀请读者尝试/思考
```

Example hook: "职业成功的最被低估技能：战略性懒惰"

---

## 3. Listicle (N个要点)

Best for: research summaries, resource compilations, methodology sharing.

```
[Hook]    "我[读了100本书/采访100人/花了X时间]，这是我的发现"
[Cred]    建立可信度 + 预告价值
[Top 1-3] 最重要的发现（详细展开，每个有案例）
[Mid 4-6] 次级发现（简短描述）
[Extra]   补充发现（可选）
[Synth]   这些要点的共同模式/主题
[CTA]     选择一两个开始尝试
```

Example hook: "我采访了100位CEO，他们都做这一件事 differently"

---

## 4. Counter-Intuitive (颠覆→重建)

Best for: provocative claims, hidden truths, concept reconstruction.

```
[Hook]    极端反常识的声明
[Disrupt] 打破读者的既有认知
[Shock]   展示问题的严重性/普遍性
[Rebuild] 提出新框架/新理解
[Evidence] 支持新框架的案例和数据
[Relief]  给读者希望和方向
[CTA]     邀请加入新认知阵营
```

Example hook: "大学完全没能教会我写作，这是我花了500小时研究大师后的发现"

---

## 5. Personal Journey (我曾经→转折→现在)

Best for: transformation stories, overcoming adversity, personal brand building.

```
[Hook]    "我曾经[某种负面状态]..."
[Low]     描述最糟糕的情况（具体、感官化）
[Trigger] 什么促使了改变？
[Process] 转变的具体步骤（不跳过细节）
[Now]     现在的状态和成果
[Reflect] 这段经历教会了我什么
[Give]    给处于同样处境的人的建议
```

Example hook: "我从20万债务到百万富翁，用了4年（没有加密货币，没有继承）"

---

## Framework Selection Decision Tree

- Have a unique personal experience? → **Narrative** or **Personal Journey**
- Have a contrarian take? → **Contrast** or **Counter-Intuitive**
- Compiled lots of information? → **Listicle**
- Want to build authority? → **Listicle** + data
- Want emotional connection? → **Narrative** + sensory details
- Story has a dramatic low point? → **Personal Journey**
- Idea challenges mainstream? → **Counter-Intuitive**
